# College Networking Portfolio

A professional repository consolidating academic and practical networking projects completed during my university journey. This portfolio spans foundational infrastructure design, deep-packet protocol forensics, and modern cloud-native VPC deployment.

---

## 📁 Projects Included

### 1. [Networking Simulation](./1-networking-simulation/)
* **Description:** End-to-end engineering, subnet planning, and deployment of an enterprise dynamic routing domain. 
* **Core Focus:** 4-router mesh diamond topology, Variable Length Subnet Masking (VLSM), OSPF route manipulation, and layer-3 MTU packet fragmentation analysis.

### 2. [Protocol Analysis](./2-protocol-analysis/)
* **Description:** Deep-packet forensic audit and behavioral analysis of fundamental network and application layer protocols.
* **Core Focus:** Wireshark hex-dump decoding, 802.1Q VLAN tagging, IP options diagnostic headers (Record Route/Timestamp), DHCP lease states, and a comparative security audit of Telnet vs. SSH.

### 3. [VPC Networks](./3-vpc-networks/)
* **Description:** Architectural design planning for a secure, highly available Infrastructure as a Service (IaaS) network in the public cloud.
* **Core Focus:** Multi-tier subnet isolation (Public/Private/DB), multi-AZ redundancy, edge transit routing via NAT/Internet Gateways, and stateful perimeter access controls (Security Groups/NACLs).
